# E-커머스 마일스톤

## 마일스톤 플로우

요구사항 분석 → 설계 문서 → 도메인 구현 (TDD) → 통합 테스트
→ 동시성 제어 → 캐싱/인덱스 튜닝 → 이벤트 기반 아키텍처 → Kafka → 부하 테스트
