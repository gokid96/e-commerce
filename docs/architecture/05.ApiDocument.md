# API 명세

## 잔액 API

### 잔액 조회

**[Description]**  
사용자 잔액을 조회한다.

**[Request]**

- URL : `GET /api/v1/users/{userId}/balance`

| Parameter | Type | Description |
|---|---|---|
| userId | Number | 사용자 ID |

**[Response]**

```json
{
  "code": 200,
  "message": "OK",
  "data": {
    "amount": 1000000
  }
}
```

| Path | Type | Description |
|---|---|---|
| code | Number | 응답 코드 |
| message | String | 응답 메시지 |
| data.amount | Number | 잔액 |

---

### 잔액 충전

**[Description]**  
사용자 잔액을 충전한다.

**[Request]**

- URL : `POST /api/v1/users/{userId}/balance/charge`

| Parameter | Type | Description |
|---|---|---|
| userId | Number | 사용자 ID |

```json
{
  "amount": 200000
}
```

| Path | Type | Required | Description |
|---|---|---|---|
| amount | Number | Y | 충전 금액 |

**[Response]**

```json
{
  "code": 200,
  "message": "OK"
}
```

## 상품 API

### 상품 목록 조회

**[Description]**  
판매 중인 상품 목록을 조회한다.

**[Request]**

- URL : `GET /api/v1/products`

**[Response]**

```json
{
  "code": 200,
  "message": "OK",
  "data": {
    "products": [
      {
        "id": 1,
        "name": "상품명",
        "price": 30000,
        "stock": 100
      }
    ]
  }
}
```

| Path | Type | Description |
|---|---|---|
| code | Number | 응답 코드 |
| message | String | 응답 메시지 |
| data.products[].id | Number | 상품 ID |
| data.products[].name | String | 상품명 |
| data.products[].price | Number | 상품 가격 |
| data.products[].stock | Number | 재고 수량 |

## 쿠폰 API

### 보유 쿠폰 목록 조회

**[Description]**  
사용 가능한 보유 쿠폰 목록을 조회한다.

**[Request]**

- URL : `GET /api/v1/users/{userId}/coupons`

| Parameter | Type | Description |
|---|---|---|
| userId | Number | 사용자 ID |

**[Response]**

```json
{
  "code": 200,
  "message": "OK",
  "data": {
    "coupons": [
      {
        "id": 1,
        "name": "쿠폰명",
        "discountRate": 0.1,
        "expiredAt": "2026-12-31T23:59:59"
      }
    ]
  }
}
```

| Path | Type | Description |
|---|---|---|
| code | Number | 응답 코드 |
| message | String | 응답 메시지 |
| data.coupons[].id | Number | 쿠폰 ID |
| data.coupons[].name | String | 쿠폰명 |
| data.coupons[].discountRate | Number | 할인율 (예: 0.1 = 10%) |
| data.coupons[].expiredAt | String | 만료일시 |

---

### 쿠폰 발급

**[Description]**

**[Request]**

|---|---|---|---|
| couponId | Number | Y | 쿠폰 ID |

**[Response]**

```json
}
```

## 주문/결제 API

### 주문 생성 및 결제 처리

**[Description]**  
주문을 생성하고 결제를 처리한다.

**[Request]**

- URL : `POST /api/v1/orders`

```json
{
  "userId": 1,
  "couponId": 2,
  "products": [
    {
      "id": 3,
      "quantity": 2
    }
  ]
}
```

| Path | Type | Required | Description |
|---|---|---|---|
| userId | Number | Y | 사용자 ID |
| couponId | Number | N | 쿠폰 ID |
| products[].id | Number | Y | 상품 ID |
| products[].quantity | Number | Y | 주문 수량 |

**[Response]**

```json
{
  "code": 200,
  "message": "OK"
}
```

```json
      {
        "id": 1,
        "name": "상품명",
        "price": 30000,
        "saleCount": 150
      }
    ]
  }
}
```

| Path | Type | Description |
|---|---|---|
| code | Number | 응답 코드 |
| message | String | 응답 메시지 |
| data.products[].id | Number | 상품 ID |
